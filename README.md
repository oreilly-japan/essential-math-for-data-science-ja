本リポジトリはオライリー・ジャパン発行書籍『[データサイエンスのための数学入門](https://www.amazon.co.jp/dp/4814401264/)』のサポートサイトです。

## 構成

```
.
├── requirements.txt      # パッケージ一覧
├── notebooks
│   ├── ch01
│   │   └── ex01.ipynb  # 1章のコード（ノートブック形式）
│   ├── ch02
│   │   └── ex02.ipynb  # ...
 ...
│   └── AppendixC
│       └── ex23c.ipynb
└── python
    ├── ch01
    │   ├── ex01-1.py    # 1章のコード（Pythonコード）
     ...
    └── AppendixC
         └── ex23c.ipynb
```

サンプルコードの解説は本書籍をご覧ください。
